# Mini Go Resto App

## Description
This application is to manage mini go restaurant

## Environment Setup

### Requirements

This project should use
* Ruby 2.6.5
* Bundler 2.1.4
* Rails 6.0.2.1

## Build Instruction
Run
1. `bundle install`
2. `rails db:create`
3. `rails db:migrate`

## Test Instruction
run `rspec`

## Run Instruction
run `rails s`
